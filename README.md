安装方法

安装jekyll   gem install jekull

如果gem无法安装，是因为国内把rubygems.org屏蔽了

使用taobao的gem 实时同步

gem source --remove https://rubygems.org/   (这里结尾的/不能省略，地址一定要打对)

gem source -a https://ruby.taobao.org/

gem source -l

sudo gem install jekyll

执行 jekull server
