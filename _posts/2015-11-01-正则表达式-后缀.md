## 正则表达式中/i,/g,/ig,/gi,/m的区别和含义

/i (忽略大小写)

/g (全文查找出现的所有匹配字符)

/m (多行查找)

/gi(全文查找、忽略大小写)

/ig(全文查找、忽略大小写)


{% highlight javascript %}
  
  var str = '#this #is__ __#a test###__';

  str.replace(/#|_/g,''); // result: "this is a test"

{% endhighlight %}
